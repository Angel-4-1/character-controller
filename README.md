
# Character controller

A React Three Fiber project using Typescript + Vite

## Installation
### With npm
```bash
  npm install
```
### With yarn
```bash
  yarn
```
    
## Deployment

To deploy this project on local environment:

### With npm
```bash
  npm run dev
```
### With yarn
```bash
  yarn dev
```

To create a build version of the project:

```bash
  npm run build
```
